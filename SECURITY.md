# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| `1.x` | Yes |

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Use GitHub private vulnerability reporting for this repository when available.
If that route is unavailable, contact the repository maintainers privately and
include:

- affected package or CDN asset
- affected version
- reproduction steps
- impact summary

We will triage reports privately and coordinate a fix before public disclosure.
